HOW TO EXTRACT
==============
Windows : keep all 1GB.z* files in the same folder, then open
          1GB.zip with 7-Zip, WinRAR, or PeaZip and Extract.
          (Windows Explorer cannot handle split zips — use 7-Zip.)

Linux   : zip -s 0 1GB.zip --out joined.zip && unzip joined.zip
macOS   : same as Linux (install zip via Homebrew if needed).
